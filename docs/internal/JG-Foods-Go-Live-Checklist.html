Phil

Critical
Phil

✓
Phil

Supabase → Authentication → Users → Invite user. Use Jon's email. He'll receive a link to set his password. This is his login for the admin app.

Critical

3
Third-party connections
Each one unlocks a set of features. Do them in order of priority.
Configuration
✓
Phil
EmailJS — set up account and wire in
Create account at emailjs.com. Add an email service (Jon's Gmail or a JG Foods address). Create two templates: order confirmation (to customer) and new order alert (to Jon). Drop the SERVICE_ID and TEMPLATE_IDs into the admin app. Unlocks: invoice emails, order confirmations.
✓
Phil
Twilio — set up account and wire in
Create Twilio account, get a UK SMS number (~£1/month). Add the Account SID, Auth Token, and sender number to a Netlify environment variable. Wire into the SMS send buttons in the admin. Unlocks: invoice texts, VIP messages, 5:30am automated reminders.
✓
Phil
Google Maps API key — route optimisation
Google Cloud Console → enable Routes API → create API key restricted to the admin domain. Drop it into the route planning section of the admin. Unlocks: optimised delivery run ordering.
✓
Phil
Claude API key — AI features via Netlify Function
Anthropic console → create API key → add to Netlify environment variables. Write a single Netlify Function (functions/claude.js) that proxies requests to the API. Unlocks: paste-to-parse order extraction, AI message redrafting, VIP text generation.
